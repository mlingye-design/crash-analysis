# Crash Analysis Project

## Overview
This project analyzes traffic crash patterns using spatial and socioeconomic data.

## Data
- Crash records
- Road network data
- Land use and demographic variables

## Methods
- Spatial aggregation
- Exploratory analysis
- Visualization of crash patterns

## Results
The analysis reveals strong spatial clustering of crashes and associations with road characteristics.
